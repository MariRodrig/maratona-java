# Exercício 45 - Cadastro de Produto

## Objetivo

Desenvolver um sistema simples de cadastro de produtos utilizando os conceitos de Programação Orientada a Objetos em Java.

## Requisitos

O sistema deve permitir cadastrar um produto informando:

- Código
- Nome
- Quantidade inicial (opcional)

Caso não exista quantidade inicial, o estoque deverá começar com **0**.

## Regras de Negócio

- O código do produto não poderá ser alterado após o cadastro.
- O nome poderá ser alterado.
- A quantidade em estoque não poderá ser modificada diretamente.
- O estoque deverá ser alterado apenas pelos métodos:
    - Entrada de estoque
    - Saída de estoque

## Funcionalidades

- Cadastro do produto
- Escolha de estoque inicial
- Entrada de produtos
- Saída de produtos
- Exibição dos dados atualizados após cada operação

## Conceitos praticados

- Encapsulamento
- Sobrecarga de construtores
- Getters e Setter
- Métodos
- Scanner
- if/else
- toString()
- Criação de objetos
- Escopo de variáveis
